# DORA-Disruption-Optimized-Risk-Assurance-by-CIRCUITSTORM
DORA is an AI-powered parametric micro-insurance platform for gig workers, enabling real-time income protection with instant, claim-free payouts. Built by Team CIRCUITSTORM for DEVTrails 2026, it uses ML and live data to detect disruptions and automate compensation via a scalable architecture.
